## 配置Dify API密钥（5分钟）

### 步骤1：获取Dify API密钥
1. 登录 Dify平台
2. 进入应用设置
3. 复制API Key

### 步骤2：更新数据库
在Supabase Dashboard执行以下SQL：

```sql
UPDATE assistants 
SET dify_api_key = '您的真实Dify_API密钥'
WHERE status = 'active';
```

### 步骤3：立即测试
刷新网页，发送消息即可看到AI回复！

## 常见问题

**Q: 发送消息后没有回复？**

**Q: 如何添加更多助手？**

**Q: 如何修改登录密码？**  
A: 更新users表中的password字段
